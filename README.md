### 1. 프로젝트 개요 👀

본 프로젝트는 스파르타 캐릭터가 메타버스와 같은 공간을 탐색하고, 맵과 상호작용할 수 있는 게임을 만드는 것 입니다.
과제로 주어진 필요기능과 추가적인 도전기능 모두를 구현하고 특색있는 기획을 추가하여 완성도있는 게임을 만드는 것이 목표입니다.
